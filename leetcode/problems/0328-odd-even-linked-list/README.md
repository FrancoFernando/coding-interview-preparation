# 328. Odd Even Linked List

**Difficulty:** Medium
**Link:** [LeetCode](https://leetcode.com/problems/odd-even-linked-list/)

## Problem Description

Given the head of a singly linked list, group all the nodes with odd indices together followed by the nodes with even indices, and return the reordered list.

## Approach

Maintain two pointers for odd and even lists, then connect them at the end.

## Complexity

- **Time Complexity:** O(n)
- **Space Complexity:** O(1)
